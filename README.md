# ollila-translator
A machine translator for a my custom language

This repository is a translation tool for ollila, a language I created.

It is almost entirely written using Vive coding.

Currently, this only support Japanese.

This translation tool specializes in pure SVO format.

It is not suitable for full-scale use.


## Ollila sample

1. 水を生きるために飲む
```
scollog dor lloy llqyawo
```

2. 未来のために本を読む
```
vayan dor oiiauq ctcoller
```

3. あなたのために食事を残す
```
mism dor csossq rem
```
